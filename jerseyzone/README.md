عيش شغفك، البس هويتك

## About
JERSEY ZONE is a premium online store for football jerseys and sports apparel in Libya. Built with Next.js 15, TypeScript, Tailwind CSS, and Supabase.

## Features
- 🎨 Modern dark theme with neon purple accents
- 📱 Fully responsive (mobile-first)
- 🛒 Shopping cart with local state
- 📦 Product categories: Club, National, Classic
- 🔍 Advanced filtering by team/country
- 💬 WhatsApp order integration
- ⚡ Fast, SEO-optimized, accessible

## Tech Stack
- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Supabase (for future backend)
